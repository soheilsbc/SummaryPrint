The code is written in C#
First you need to click on the project tab on Visual Studio and then click on NuGet Package Manager.
Then install Newtonsoft.Json
Please put the path for the JSON files in the Program.cs file where it is stated and commented. 
Then run the project and it automatically stream the data and print the summary. 
It shows based on the steps in the task. First it list the users in each year, then median for the number of friends, median for age, mean for balance and mean for the number of unread messages for active females.
